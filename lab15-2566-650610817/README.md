# Lab 15 : Mantine Marathon Register form

หลังจากการ Fork และ Clone repository แล้ว, ให้เปิดโฟลเดอร์ด้วย VSCode และรันคำสั่งใน terminal:

```bash
npm install
npm run dev
```
